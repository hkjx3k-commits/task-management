## 1. プロジェクト概要

Trello風のタスク管理アプリ。ボード・リスト・カードの3階層構造で個人タスクを管理する。  
サーバーレス・認証なし。データはlocalStorageに永続保存。

---

## 2. 技術スタック

| 役割 | ツール | バージョン（目安） |

| ドラッグ&ドロップ | @dnd-kit/core + @dnd-kit/sortable | 6.x |

## 4. データ型定義

```typescript
// カード（個々のタスク）
type Card = {
  id: string;
  title: string;
  memo: string;        // メモ（自由記述）
  dueDate: string;     // 期日（ISO 8601形式: "2026-05-10"）
  createdAt: string;   // 作成日時（ISO 8601形式）
};

// リスト（列）
type List = {
  id: string;
  title: string;
  cards: Card[];
};

// ボード（全体）
type Board = {
  id: string;
  title: string;
  lists: List[];
};
```

## 6. 機能要件（詳細）

### 6-1. ボード

| ID | 機能 | 詳細 |
|----|------|------|
| B-1 | 表示 | 起動時にlocalStorageからデータを読み込む。なければ初期データを生成 |
| B-2 | タイトル編集 | クリックでinput表示、blur/Enterで確定 |

### 6-2. リスト

| ID | 機能 | 詳細 |
|----|------|------|
| L-1 | 追加 | 末尾に新しいリストを追加。タイトルは「新しいリスト」で初期化 |
| L-2 | 削除 | 確認ダイアログ後、リストとその中のカードをすべて削除 |
| L-3 | タイトル編集 | クリックでinput表示、blur/Enterで確定 |
| L-4 | 初期データ | 初回起動時：「やること」「進行中」「完了」の3リストを生成 |

### 6-3. カード

| ID | 機能 | 詳細 |
|----|------|------|
| C-1 | 追加 | リスト末尾に追加。タイトルは「新しいカード」で初期化 |
| C-2 | 削除 | カード詳細モーダルから削除 |
| C-3 | タイトル編集 | カード詳細モーダルで編集・保存 |
| C-4 | 移動 | @dnd-kit でリスト間・リスト内のドラッグ&ドロップを実装 |
| C-5 | 詳細モーダル | カードクリックで表示。タイトル・メモ・期日を編集できる |
| C-6 | 期日の色分け | 期日 < 今日 → 赤背景、期日 = 今日 → 黄背景 |

### 6-4. データ保存

| ID | 機能 | 詳細 |
|----|------|------|
| D-1 | 自動保存 | Boardステートが変化するたびlocalStorageへJSON.stringify保存 |
| D-2 | 読み込み | 起動時にlocalStorageからJSON.parseで復元 |

## 7. ドラッグ&ドロップ設計

@dnd-kit を使用する。

```
DndContext（最上位）
└── SortableContext（各リスト）
    └── useSortable（各カード）
```

- **同一リスト内の移動**：`arrayMove` で並び替え
- **リスト間の移動**：`over` のIDを判定し、対象リストへカードを移動
